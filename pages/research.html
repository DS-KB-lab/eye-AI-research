Research plan

The near-term goal is not to be exhaustive. It is to build a rigorous, expandable ophthalmic AI resource that starts with the strongest evidence base and grows by verified additions.

1. Start with foundation models

This is the best-populated and most mature track, with anchor works such as RETFound and VisionFM.

2. Expand with careful topic logic

Visual grounding should be interpreted broadly; PEFT should include LoRA, adapters, and prompt tuning; hyperbolic methods should begin as a method page.

3. Prefer official links

Journal pages, DOI pages, official conference pages, and author-maintained repositories outrank secondary summaries.

4. Keep the schema automation-ready

The dataset is intentionally shaped so future cron jobs can add papers, notes, figures, and update logs without redesigning the site.
