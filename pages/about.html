About this site

Eye AI Research is designed as a public-facing, version-controlled ophthalmic AI reading surface. The emphasis is on useful curation rather than maximal volume.

Public by default

Everything is organized to publish cleanly on GitHub Pages.

Conservative metadata

Uncertain venue or highlight information is labeled gently instead of guessed aggressively.

Automation-ready

The site structure and JSON schema are ready for future scheduled updates.

Figure-first reading

Every paper card reserves space for a core model diagram so the site can become visually scannable over time.

Built for iterative expansion, not one-shot completeness.
